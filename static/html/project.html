App - Dishes Need Packing

My first published product is a free-to-play Android app found on the Google Play Store. It is a fast-paced puzzle arcade style game. I've put a fun twist on a boring house hold chore, packing the dishwasher.

It comes fully loaded with 100 levels, a custom game mode, and a backend server to store user highscores.

Find the App Here on the Google Play Store

Software - Top 10 Artists of the Month

An application that displays info and stats of a users top 10 favorite music artists or bands of each month. I've maintained and updated this program for 6 years now! Back when I started working on it I created my own text-file based database to handle the data. I have a number of interesting ways to display the information. I'm still using this software today to track my favorite music, it has nearly 6 years of data stored.
